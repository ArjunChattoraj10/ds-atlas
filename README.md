# DS Atlas - The Data Science Quick-Reference Guide

**A quick-reference for Data Science methods.**
What it does. When to use it. In plain English.

🔗 [dsatlas.vercel.app](https://dsatlas.vercel.app)

---

## What is DS Atlas?

DS Atlas is a fast, searchable reference guide for Data Science methods and techniques. Each method is broken down into plain-language explanations, pros and cons, inputs and outputs, assumptions and practical notes, so you can quickly decide whether a technique fits your problem, without digging through textbooks or documentation.

It covers methods across Machine Learning, Statistics, Econometrics, Generative AI, NLP and more, organized by the task you're trying to accomplish rather than by academic taxonomy.

---

## Features

- **Task-first navigation**: browse by what you're trying to do (predict a continuous outcome, handle imbalanced classes, reduce dimensionality, etc.)
- **In Plain English**: every method has a plain-language explanation, no jargon required
- **Pros & Cons**: structured trade-off summaries to help you choose between methods
- **Inputs / Outputs / Assumptions / Notes**: the practical spec for each technique
- **Tag filtering**: filter methods by discipline (ML, Stats, Econometrics, Bayesian, NLP, etc.)
- **Smart Search**: intent-aware ranked search with typo-tolerance, synonym expansion, and contextual relevance cues
- **Tiles / List toggle**: switch between browsing modes
- **Light / Dark mode**: because obviously

## Running Locally

```bash
# Clone the repo
git clone https://github.com/ArjunChattoraj10/ds-atlas.git
cd ds-atlas

# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be available at `http://localhost:5173`.
